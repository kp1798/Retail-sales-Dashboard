# Retail Sales Dashboard

## 📊 Project Overview
This Power BI project analyzes retail store sales performance using transactional data.  
The dashboard highlights revenue trends, product category performance, and customer purchasing behavior.  
It was inspired by JJ Power BI’s Retail Sales Dashboard tutorial, adapted with custom insights and visuals.

## 🛠️ Tools & Technologies
- Power BI (data visualization, DAX measures)

## 📈 Key Insights
- Identified top‑performing product categories and regions.  
- Visualized monthly and yearly sales trends.  
- Highlighted customer segments contributing most to revenue.  
- Provided actionable recommendations for inventory and marketing.
